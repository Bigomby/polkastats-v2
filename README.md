# PolkaStats v2

> Polkadot Kusama CC2 network statistics

## Build frontend

``` bash
# Clone this repo
$ git clone https://github.com/Colm3na/polkastats-v2.git
cd polkastats-v2

# Install dependencies
$ npm install

# Serve with hot reload at localhost:3000
$ npm run dev

# Build for production
$ npm run build

# generate static project
$ npm run generate
```
